Stuff I made which I consider kinda cool. Making this section has made me realise how many wips I have done and that I haven't really finished many things to make public. So maybe this will help to motivate me to change that.

Anyways, right now I have projects I believe have done pretty well in their respective communities, so I will showcase them here.

GD Song Verifier

Quickly check if a song is usable in NG to create a level.

When I made this, I was surprised no one had made it before honestly. I believe it is a really useful tool for those who create levels, since it is quite annoying to have to go to the game for each song.

Now it has more than 300 current users, which I'm really happy with, even though I believe it could be even more if I had shared it more but I don't really feel like it lol.

Milk Filter

Choose any image you like and convert it to something you would find in the series Milk inside/outside a bag of milk!

This started as a project for myself but then realised I could actually just work a little bit more to make it public. I think this is the first thing I made a GUI for and not just a CLI program.

Really happy with this one, it had a lot of support! It has 34 starts right now, and more than 650 downloads!!!. I just find that crazy really.
